01

Идея бренда

Brand idea

Имя собрано из двух клавиш. Ctrl — инженерный контроль: точность, системность, управляемая сложность. Zen — спокойствие результата: тишина, минимум, ничего лишнего. Бренд живёт в тёмном «терминальном» пространстве, где из шума частиц собирается порядок.

^ Ctrl

Инженерия. Сетки, моноширинный шрифт, HUD-разметка, технические аннотации, версии и индексы.

○ Zen

Дзен. Пустота как материал: много чёрного, один смысловой центр на экран, медленный спокойный ритм.

02

Логотип

Logo

Логотип — две клавиши: Ctrl с символом «^» и Zen с символом «○» (энсо, круг дзен). Шрифт знака — Consolas. Основная среда — тёмная; светлая версия используется на бумаге и светлых носителях.

Мин. размер · экран 120 px ширина полного лого; знак — от 24 px
Мин. размер · печать 25 mm ширина полного лого; знак — от 6 mm
Геометрия клавиши R = 6.25% радиус угла = 1/16 стороны; обводка = 1/58 стороны

03

Цвет

Color

Палитра строго монохромная — как терминал. Цвет допускается только внутри контента (скриншоты продуктов, арты), никогда — в элементах айдентики и UI-хроме.

{{ c.name }} {{ c.hex }} {{ c.use }}
Пропорции · Proportions
VOID 72%
CARBON 18%
PAPER 10%
  • —Белый — дефицитный ресурс: не более ~15% площади экрана, он обозначает главное.
  • —Текст на Void/Carbon — Paper (контраст ≥ 15:1) или Ash для вторичного (≥ 4.6:1, AA).
  • —Акцент Signal — единственный цвет системы. Только для микроэлементов: мигающий курсор, индексы разделов, маркеры «—», статусы. Не более ~1% площади экрана.
  • —Signal никогда не используется для заливок кнопок, крупного текста, иконок и логотипа — они всегда монохромные.
  • —Никаких цветных градиентов. Глубина — только светом: прозрачности белого 4–13%.

Веб и вёрстка

Frontend rules 
:root {
  /* цвет */
  --cz-void:    #0A0A0B;  /* фон страницы */
  --cz-carbon:  #161618;  /* карточки, клавиши */
  --cz-line:    #2A2A2C;  /* границы 1px */
  --cz-ash:     #8A8A8E;  /* вторичный текст */
  --cz-paper:   #F5F4F0;  /* текст, лого */

  /* шрифт */
  --cz-mono: 'JetBrains Mono', Consolas, monospace;
  --cz-sans: 'IBM Plex Sans', sans-serif;

  /* радиусы: от клавиши (R = 1/16 стороны) */
  --cz-r-s: 8px;  --cz-r-m: 12px;  --cz-r-key: 16px;

  /* сетка: 8pt; отступы секций 96px */
  --cz-gap: 24px;

  /* моушн */
  --cz-fast: 120ms; --cz-base: 240ms; --cz-slow: 480ms;
  --cz-ease: cubic-bezier(0.2, 0, 0, 1);
}
  • —Сетка 12 колонок, контейнер 1180px, все отступы кратны 8.
  • —Границы — 1px solid var(--cz-line). Тени не используем: глубину даёт тон поверхности и границы.
  • —Кнопки — «клавиши»: carbon-фон, радиус --cz-r-key, при нажатии утапливаются на 2px.
  • —Фокус — 1px dashed var(--cz-ash), offset 3px. Никаких синих outline.
  • —Фактуры из раздела 05 — только CSS-градиентами, не картинками.

10

Моушн

Motion

Движение — спокойное и механическое, как отклик клавиатуры. Ничего не «пружинит» и не летает: короткие точные переходы, дискретные шаги вместо плавных морфов. Всегда уважаем prefers-reduced-motion.

--cz-fast 120ms · ховеры, нажатия --cz-base 240ms · переходы состояний --cz-slow 480ms · появление секций easing cubic-bezier(0.2, 0, 0, 1)

Лого из частиц

Particle dissolve · live
